Seguridad de Django

Modo Debug y Hosts Permitidos

SECRET_KEY No editable desde la web

Por seguridad, la SECRET_KEY no se muestra ni se puede modificar desde esta interfaz. Cualquier acceso al DOM del navegador podría exponerla aunque esté enmascarada.

Para cambiarla, edita directamente el archivo .env en el servidor y reinicia el proceso.
Ejemplo: nano /ruta/del/proyecto/.env → modifica SECRET_KEY=...

SIEMPRE en False en producción.

Separa múltiples hosts con coma. Usa * solo en desarrollo local.
